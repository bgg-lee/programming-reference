## 사용 방법

1. 이 레포지토리를 클론합니다:
    ```bash
    git clone git@github.com:GoGiants1/GSDS-Qual.git
    ```
2. 문제 폴더로 이동하여 문제를 풉니다:
    ```bash
    cd algorithm-problems

3. 문제를 풀고, 해답을 제출합니다:

    ```
